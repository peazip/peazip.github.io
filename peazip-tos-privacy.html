Cookies informative


This document provides for detailed information on the use of cookies, on how they are used on this website and how to manage them.
Please note that this website does NOT use cookies.


Definitions

Cookies are small text files that are stored by a web server on the of the storage of the device used (computer, tablet, smartphone), depending on the users’ browser options. This information can be used during the same online session or when accessing again the same website even after several days.
Duration (either limited or unlimited) of the cookie is set when the cookie is created: if information is kept past the duration of the session the cookie is defined Permanent.

Based on their characteristic and use, cookies can be classified as:
  1. Technical: cookies necessary for the correct functioning of a website, with duration strictly limited to the duration of the session
  2. Analytic: cookies used to collect anonymous aggregated data on user’s browse experience and use of services. They can be deselected with no loss of functionality.
  3. Profiling: cookies that identify (also on aggregate anonymous basis) user’s preferences to ensure a better browsing experience and/or ads matching user habits and expectations.
User’s browser can receive cookies that are created and managed from a website (third party) different from the one that the user is visiting, this type of cookie is defined Third Party. The use of the information collected by third parties with their cookies is regulated by their respective informative notes and privacy policies.

Many online sources provides more in-depth technical and legal information about cookies and other similar technologies. Wikipedia cookie article is a good starting place for accessing a multilingual compendium which takes in account different international legal regimes, another good source of information is http://www.aboutcookies.org/.


Which cookies are used by this website, for what purpose, and how to selectively allow or disallow them

PeaZip website does not use cookies of any type, in the best interest of user's privacy.

To guarantee users the verifiability of the enforcement of this rule of conduct, the source code of the website in all of its components is publicly available online on GitHub repository https://github.com/peazip/peazip.github.io and is entirely auditable at any time by any subject.
Also, GitHub, which hosts PeaZip website, has itself a very restrictive policy about cookies: https://github.blog/2020-12-17-no-cookie-for-you/

Privacy notice



PeaZip software

PeaZip software does not collect, process or communicate/diffuse personal identifiable information about users.

PeaZip, in the best interest of users' privacy, lacks any function to store and send data of any kind from user's computer, the only networking capability being linking the online resources (pages of updates, support, issue tracker, etc).

From Options > Settngs, General tab, Privacy and Security group it is possible to access to all relevant features from a single place, as:
  • verify updates
  • check for available documentation, issue tracker, CVE entries
  • selectively reset temporary work files, configuration, history, bookmarks, etc
  • define a custom work folder
  • disable history, etc
The software is distributed thorugh GitHub, either in form of downloadable packages for different platform and of Git repository, in any case when using GitHub repository to access to ther software refer to GitHub Privacy Policy https://docs.github.com/en/github/site-policy/github-privacy-statement

PeaZip project software is Open Source, which means it can be legally hosted and distributed under LGPLv3 licensing terms by any third party actor: when using third parties sources to get our software in any form please refer to their own Privacy Policies.


PeaZip website

PeaZip project website is a personal website, and does not collect, process or communicate/diffuse personal identifiable information about visitors.

The website is hosted on GitHub, which logs visitors' technical information for the purpose of maintaining and improving the quality and security of its hosting infrastructure: GitHub does not share any of the collected information with (nor otherwise provide access to) the author of the website.
