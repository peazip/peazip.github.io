How to find more support information


DEVELOPER'S EMAIL

The best way to obtain helpful information about operating PeaZip free archiver utility is searching in the PeaZip online tutorial, or the most recent version of the downloadable offline help file (PDF) - which is part of standard PeaZip packages, and to read in-depth single purpose mini-tutorials and how to pages in Frequently Asked Questions section.

How to post a support request

 For topics and issues not covered by the documentation and manuals, the preferred way to get support is using SourceForge Tickets, in order to rationally search for existing issue reports, and to file program bugs, new features requests, and support requests, to submit code patches etc.
When posting to report issues to tracker database, or mailing, please try to be more informative and detailed as possible, i.e. specify PeaZip's version and package, host operating system details (language, 32 or 64 bit, Windows' version and service pack level, Linux distribution, version, KDE or Gnome, etc), the complete way to replicate a problem and so on.
Informative and detailed posts not only makes easier to find solutions to problems and to get a comprehensive response, but can also become a precios source of information for other users.

A quick way to find relevant answers in published issue reports is using search integrated in the bug tracker / feature request database, which is linked on the top of this page.
