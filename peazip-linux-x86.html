Linux WinRar WinZip replacement, rar, zip file archiving utility

PeaZip file archiver utility works with most common archive types, open and extract 7Z, ACE(*), BZ2, GZ, RAR, TAR, ZIP, ZIPX archives, installers (DEB, RPM), disk images (DMG, ISO, UDF), and Microsoft WIM Windows Imaging Format, cutting edge experimental formats like PAQ family (multiple times Hutter prize winner) and new FreeArc's ARC (strong encryption, very efficient data compressor). PeaZip for Linux zip and unzip free archive formats as well as proprietary ones (unace, unrar).

PeaZip for Linux x86 provides the same wide array of file archiving, compression algorithms, strong encryption (AES, Serpent, Twofish), file management features (secure data deletion, compute checksum/hash of files), and uses same intuitive graphical user interface on Microsoft Windows, macOS and on various flavours of Linux and BSD, supporting Gnome, KDE, XFCE, and other desktop managers.

Like Ark and FileRoller archivers, PeaZip file archiver works as frontend application for Open Source components from 7-Zip / p7zip archiver and other great projects (FreeARC, PAQ, UPX...), providing an unified GUI file archiver tool that represents a Free Software replacement for WinRar, WinZip and similar software utilities, with the benefit of being natively portable, cross-platform, independent from widget set and window manager.



PeaZip for Linux x86 download notes


PeaZip for Linux x86 is recommended for legacy systems, while x86-64 and aarch64 builds are recommended for modern Linux machines.
GTK2 version packages may require to install some components of gtk/gdk libraries, as i.e. libgdk_pixbuf, see full list of dependencies for GTK2 packages.
Qt version packages requires Qt 4.3.4 or more recent, and needs libqt4intf.so and libqt4pas5 installed; it is recommended you install most up to date missing libraries using distribution-specific methods, i.e. apt-get install libqt4pas5:i386. Read full list of dependencies for Qt packages.

 (*) To open and extract ACE requires to download and install UNACE Plugin for Linux, based on proprietary WinAce unace code (compiled for Linux platform), that being incompatible with Open Source distribution model is provided as separate package, see unace / unrar Add-ons page.
RAR files extraction is supported out of the box, including new RAR5 format, using OpenSource unrar implementation from 7-Zip / p7zip project.
On Add-ons page it is featured UNRAR5 Plugin for Linux, based on (WinRar) RarLab's unrar code, that is not compatible with Open Source distribution model and is provided as separate download: it is made avaliable as optional replacement to Open Source p7zip's unrar tool to open and extract rar files in the unlikely case the Free Software component encounters problems.


freedesktop_integration folder in (peazip)/res/batch directory contains script examples for integrating PeaZip Portable versions in the host system, and to enhance / customize integration of installable packages.
