Free RAR TAR ZIP files extractor, unzip utility for Windows

PeaZip is a free archiver software providing powerful multi format archive manager and complete file explorer.

As archive extractor PeaZip  can unzip 200+ archive formats: 001, 7Z, Brotli, ACE(*), ARJ, CAB, DMG, ISO, LHA, RAR/RAR5, R01, UDF, ZIP, ZIPX, Zstandard...

As for archive creation, PeaZip free archiver compress files to 7Z, ARC, BR, BZ2, GZ, PAQ, PEA, RAR(**), sfx, TAR, UPX, WIM, XZ, ZST and ZIP / ZIPX format.

This software offers full read / write support for multi-volume spanned (split) archives, and encrypted (password protected) data.

(*) Extraction of ACE archives requires separate unace plugin available on PeaZip add-ons page as separate download due the closed source nature of the ace extractor tool, based on WinAce code.
(**) PeaZip can create RAR archives if WinRar is installed in the system, autonomous support to rar compression is not possible due to rar licensing limitations. Please nore that extraction of RAR archives is natively supported, separate unrar plugin is provided solely as alternative royalty free rar extractor engine.


Compress, open, extract, encrypt, backup, split files...

PeaZip free archiver is focused on security, supports strong encryption, two factor authentication (password and key-file), integrate encrypted password manager and secure delete functions for best protection of your online and offline data privacy.
Depending on the chosen archiving format, PeaZip can encrypt / decrypt archives using AES, Serpent, Twofish, and Blowfish ciphers.

Find duplicates, verify hash / checksum, archive file converter, save backup scripts, split and merge files, and many other handy features are provided to manage files with ease, alongside compress / extract functions for supported archive types.

PeaZip supports drag & drop extraction / archiving, full integration with Windows context menu, provides user friendly GUI relying on solid, proven Open Source technologies of 7-Zip, p7zip, Brotli, FreeArc, PAQ, Zstandard projects.


PeaZip for Windows 32 bit download notes


PeaZip 32 bit is intended for legacy systems, while it can run on Windows 64 bit systems it is slower than PeaZip 64 bit, which should be preferred on modern Windows devices - ther same holds true for the Portable version.
The scope of PeaZip 32 bit is retrocomputing, supporting legacy 32 bit Windows systems that would be unable to run PeaZip 64 bit, going back to Windows 2000 and Windows XP, 32 bit versions of Vista and Windows 7, 8, 8.1, 10, and any emulator / compatibility layer capable of running WIN32 executables.
This version of PeaZip free archiver utility requires Win32 support, any Microsoft Windows system for desktop PC or tablet, or compatible API as Wine and ReactOS.
